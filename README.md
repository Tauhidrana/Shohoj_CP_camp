# Shohoj CP Camp

Landing page for **Polytechnic CP Camp 1.0** — a free, beginner-friendly Competitive Programming bootcamp by [Shohoj Coding](https://cp-shohoj.vercel.app) for polytechnic students in Bangladesh.

**Live site:** https://cp-shohoj.vercel.app

## About the camp

কোথা থেকে শুরু করবে বুঝতে পারছ না? এই বুটক্যাম্প একদম শূন্য থেকে Problem Solving, Logic Building এবং CP-র ভিত্তি গড়ে দেয় — সম্পূর্ণ ফ্রিতে।

| Detail | Info |
|--------|------|
| Start date | 20 June 2026 |
| Duration | 1 month |
| Level | Beginner |
| Fee | Free |
| Audience | Polytechnic students only |

**Register:** [Google Form](https://docs.google.com/forms/d/e/1FAIpQLSeKAz_us_VhjeOUBmQAVRiUr01J1W5TqDuzZSIg7eulipPi4A/viewform)

> Non-polytechnic applicants are automatically excluded during selection.

## What's included

- **Live interactive classes** — ask questions in real time, not just recorded videos
- **Topic-based practice contests** — test progress on real judges after each topic
- **Codeforces & CodeChef guidance** — from account setup to joining contests
- **Curated problem lists** — ordered by topic so you know what to solve next
- **Beginner-friendly environment** — no question is treated as “too basic”
- **CP roadmap** — a clear path to keep learning after the camp ends

## Camp journey

1. **Registration & setup** — form, Codeforces/CodeChef accounts, editor setup
2. **Basics & logic** — programming fundamentals and logic building from scratch
3. **Problem solving** — topic classes, problem lists, and practice contests
4. **Real contests** — finish the camp ready to join rated contests

## Local preview

Open `cp-camp.html` or `index.html` in a browser, or serve locally:

```bash
npx serve .
```

Then visit `http://localhost:3000`.

## Deploy to Vercel

```bash
npx vercel --prod
```

Or connect this repository in the [Vercel dashboard](https://vercel.com/new) for automatic deploys on every push to `main`.
